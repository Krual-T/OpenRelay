# Requirements

## Goal
把“过一段时间回来快速看懂发生了什么”做成飞书端的一等体验。

## Problem Statement
transcript 富文本渲染已经明显提升可读性，但异步回看仍缺少稳定摘要、停止原因、下一步建议和历史会话快速预览。

## Required Outcomes
1. 完成正式设计稿，明确异步回看的状态摘要、关键动作摘要、停止原因、下一步建议和历史恢复入口。
2. transcript 或相关卡片能稳定提供“发生了什么/为什么停下/建议下一步”的最小摘要闭环。
3. 历史会话列表或恢复入口能支持快速判断“要不要继续这条线”。

## Non-Goals
- 不在 package migration 这一步直接实现功能代码。
- 不在需求未澄清前提前透支更复杂的交互或产品面。

## Constraints
- 当前 package 是从历史遗留任务描述提炼出的正式设计入口。
- 后续实现前仍应补充更细的总体设计和详细设计。
